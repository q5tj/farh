## 🔍 التحقق من أن كل شيء يعمل

### تحقق من قاعدة البيانات

بعد أول عملية دفع اختبارية، شغّل هذا الأمر في Supabase SQL Editor:

```sql
-- تحقق من جدول المدفوعات
SELECT 
  id,
  kind,
  status,
  amount_halalas,
  moyasar_id,
  created_at
FROM public.payments
ORDER BY created_at DESC
LIMIT 1;

-- النتيجة المتوقعة:
-- status = 'paid'
-- moyasar_id = معرّف من Moyasar
-- amount_halalas = المبلغ
```

### تحقق من الحجز

```sql
SELECT 
  id,
  payment_status,
  deposit_paid_at,
  status
FROM public.bookings
WHERE user_id = (SELECT auth.uid())
ORDER BY created_at DESC
LIMIT 1;

-- النتيجة المتوقعة:
-- payment_status = 'paid'
-- status = 'confirmed'
-- deposit_paid_at = الآن تقريباً
```
